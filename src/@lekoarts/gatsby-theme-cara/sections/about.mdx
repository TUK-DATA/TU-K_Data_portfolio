## About

The project is meant to create a free centralised data access for Software Developers, Data Scientists and other data enthusiast around the school. The data can be used to create visualizations, for predictive analysis or just have fun with it.

For those would want to contribute to the project or join our **[Github](https://github.com/TUK-DATA)** organization, reach out through the contacts below.
